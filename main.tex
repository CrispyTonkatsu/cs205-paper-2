\section{Introduction}
The power to analyze relationships between large amounts of data has never been as accessible as it has been with the development of machine learning algorithms. However, there is only so much data in the world so companies that want to push what their products can do rely on large unmonitored sets of data. This can cause algorithms to be biased towards minority groups. This issue exponentiates when the usage of this technology now stretches from healthcare, recruitment, law enforcement and more. This essay will attempt to focus on salient issues brought up by the 2020 documentary "Coded Bias".
